## 🚀 如何使用这个仓库？

### **第1步：每周的学习流程**

**星期一到星期五：** 学习新知识
1. 打开对应的笔记文件（比如 `01-网络基础.md`）
2. 认真读一遍（不用全部记住！）
3. 用自己的话，在 `学习日记.md` 里写下来

**星期六：** 做练习题
1. 打开 `在线练习平台.md`
2. 选一个平台做题
3. 把做题的心得写下来

**星期日：** 复习
1. 看看这一周学的东西
2. 把不懂的写在 `遇到的问题.md`
3. 休息！

### **第2步：更新你的笔记**

学到新东西？有新的想法？
1. 点击 ✏️ **编辑按钮**
2. 添加你的内容
3. 点击 **Commit changes** 保存

### **第3步：记录你的进度**

在 `学习日记.md` 里写：
```
# 我的学习日记

## 2026年3月12日 星期四
### 今天学了什么？
学习了网络的基础概念：
- IP地址是什么
- 为什么需要DNS
- 怎样从网站下载信息

### 我最大的收获
原来输入网址后，计算机要做那么多工作！

### 我还不懂的地方
DNS的工作流程还有点模糊
```
