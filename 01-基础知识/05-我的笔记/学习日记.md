## 💡 做笔记的小技巧

### 好的笔记应该这样写：

❌ **不好的方式：**
```
OSI模型有7层。
第1层是物理层。
...
```

✅ **好的方式：**
```
OSI模型 = 网络的7层蛋糕

为什么叫7层？
因为网络有7个重要的工作层次，从下到上分别是：
1. 物理层 - 网线、光纤等
2. 数据链路层 - 交换机
...

我的理解：
就像做一个项目，需要基础、中间层、最上面的完成品。
网络也一样。
```

### 做笔记的黄金法则：
1. 📝 **用自己的话** 解释，不要复制粘贴
2. 🎨 **画图** 帮助理解
3. 💭 **写你的想法** 和疑问
4. 🔗 **找关联** 联系到你已知的东西
5. ⚡ **简短有力** 不用写长篇大论
